I am planning on working on a way to set up the install directory much better than how it is now for those of you that wish to use a different directory name.

Setup.bat does automatically generate the path for the Documents\AutoHotkey.ahk file and you can have it semi-dynamic or static.. Edit the setup.bat file
to change the folder-name and the AutoHotkey.ahk file will be created and properly positioned ( You will need to go ahead and move the AcecoolAHK_Framework\
folder to where you want it so the symbolic link can be set-up - I may use this step to automatically generate a full path at a later date.. we'll see )..

Setup.bat is pretty self-explanatory in terms of what to edit, and what not.. When you run it it will also show you the full-paths before any changes are made.
If you don't want to make changes, click the X to close the command prompt instead of "pressing any key" to continue...
