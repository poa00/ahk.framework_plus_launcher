    ☼ Default HotKey Behavioral Alterations
    Keys Result
    Move Up One Directory
    Backspace Reverts the behavior of the backspace key to that of Windows XP where it goes up a directory instead of the new behavior of going back in history...
    Desktops & TaskView System
    ALT + Tab

    WIN + Tab    		 In Windows 10, WIN + Tab shows a beautiful layout where all of your windows can be seen, you can create new desktops, remove them, switch to them and move applications to them... With behavior modification, ALT + Tab does the same thing rather than opening the small application switcher..
    CapsLock Avoidance System
    CapsLock

    CapsLock * 2

    Shift    		 Capslock Behaves like Shift, Double Tap for CapsLock

    Hold CapsLock to type in Uppercase; On Release CapsLock is disengaged.
    Double-Tap CapsLock to engage CapsLock
    Tap CapsLock or Shift to disengage CapsLock..

    ☼ HotKeys
    Keys Result
    Toggle Hidden File Visibility
    WIN + H Toggles whether or not hidden files can be seen while using Explorer...
    Screenshot Active Window
    ALT + F12 Saves Active Window Screenshot to "Dropbox\Screenshots\"

    If you click on the toast-notification it'll open the image and ask if you want to make it public which moves it to Dropbox\Public\Screenshots\
    Cycle Audio IO Devices
    Pause / Break Desktop Speakers & PS3 Eye Microphones for the first device, Plantronics Headset for the second, Logitech Headset for the third...

    ☼ Special Character HotKeys Toggle
    Keys Result
    Toggle Spanish / German Characters
    WIN + F8 Toggle between Spanish and German Overlapping Characters

    For characters that overlap ( ALT + a o and u ) for German and Spanish, you can toggle which sub-set you want to use with this hotkey.

    WIN + I Converts as many characters to super-script as possible

    0123456789+-=(n) abcdefghijklmnopqrstuvwxyz

    ⁰¹²³⁴⁵⁶⁷⁸⁹⁺⁻⁼⁽ⁿ⁾ ᵃᵇᶜᵈᵉᶠᵍʰⁱʲᵏˡᵐⁿᵒᵖxʳˢᵗ ᵘᵛʷˣʸᶻ
    SubScript
    WIN + O Converts as many characters to sub-script as possible

    0123456789+-=(n) abcdefghijklmnopqrstuvwxyz
    Becomes
    ₀₁₂₃₄₅₆₇₈₉₊₋₌₍n₎ ₐbcdₑfghijklmnₒpqᵣstᵤᵥwₓyz

    WIN + P Converts .s to ,s and vice versa ( For German to English Currency )

    100,000,000.26464 AND 54.654.464.465.444,567

    100.000.000,26464 AND 54,654,464,465,444.567
